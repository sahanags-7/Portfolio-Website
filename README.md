## 📌 Description

This is a responsive Full Stack Personal Portfolio Website built using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB.

The website includes:

- Home
- About
- Skills
- Projects
- Resume
- Contact Form

The Contact Form stores user messages in MongoDB through a Node.js and Express.js backend.

## ⚙️ Installation

### Install Backend Dependencies

```bash
cd server
npm install
```

### Create Environment File

Create a `.env` file using `.env.example`.

Example:

```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
```

### Run the Server

```bash
npm start
```

Open the frontend using **VS Code Live Server**.
